# Contributing

Thank you for considering contributing to Statamic Heroicons!

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `composer install`
3. Run tests: `composer test`
4. Run linting: `composer lint`

## Pull Requests

- Create a feature branch from `main`
- Follow the existing code style (enforced by Laravel Pint)
- Add tests for new features
- Update the CHANGELOG.md with your changes
- Submit a pull request

## Reporting Issues

When reporting issues, please include:

- PHP version
- Laravel version
- Statamic version
- Steps to reproduce
- Expected vs actual behavior

## Security Vulnerabilities

Please report security vulnerabilities directly via email to stefan.galescu@gmail.com rather than opening a public issue.
